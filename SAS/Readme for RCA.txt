Readme for RCA/SAS

Updated August 9, 2022

This folder contains two SAS programs:

RelComp Macros C&C 98 example.SAS replicates the data, analyses, and tables presented in 

Cliff, N., & Caruso, J. C. (1998). Reliable component analysis through maximizing composite reliability. 
Psychol Methods, 3(3), 291-308.


RelCom.SAS is the general use reliable components analysis macro.
